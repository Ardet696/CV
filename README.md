# Victor Magnani — CV

Static personal CV website built with HTML5, CSS3, and vanilla JavaScript.
Hosted publicly and embedded in my [LinkedIn profile](https://www.linkedin.com/in/victormagnani/).

## Stack

- **HTML5** — semantic markup
- **CSS3** — custom grid layout and responsive styles
- **JavaScript** — hover interactions, date reveals, and contact form with input validation
- **FormSubmit.co** — serverless email delivery for the contact form

## Features

- Hover over any CV entry to reveal its date
- Click the profile photo to toggle a larger view
- Contact form with client-side validation (length limits, email format, injection prevention) that delivers messages directly to my inbox
